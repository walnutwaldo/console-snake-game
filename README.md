# Console Snake Game

## Description ##
Originally forked from https://github.com/Contagious06/console-snake-game

## Compiling ##

### Linux ###
1. wget https://raw.githubusercontent.com/Contagious06/console-snake-game/master/src/snake.c
2. gcc snake.c –lm –o snake.out
3. chmod +x snake.out
4. ./snake.out
